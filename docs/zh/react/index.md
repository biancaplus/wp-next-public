# React 组件

`@wp-next/react` 提供在 Next.js 中渲染 WordPress 内容的组件与工具。

## 安装

```bash
npm install @wp-next/react @wp-next/core
```

## 模块一览

| 模块 | 说明 |
|------|------|
| `ContentRenderer` | 带处理器管线的 HTML 渲染 |
| Processors | `imageProcessor`、`codeProcessor`、`linkProcessor` |
| SEO | `createPostMeta`、`ArticleJsonLd`、`BreadcrumbJsonLd` |
| Sitemap | `renderSitemapXml`、`postsToSitemap`、`termsToSitemap` |
