# React Components

`@wp-next/react` provides components and utilities for rendering WordPress content in Next.js.

## Installation

```bash
pnpm add @wp-next/react @wp-next/core
```

## What's Inside

| Module | Description |
|--------|-------------|
| `ContentRenderer` | Render WP HTML with processor pipeline |
| Processors | `imageProcessor`, `codeProcessor`, `linkProcessor` |
| SEO | `createPostMeta`, `ArticleJsonLd`, `BreadcrumbJsonLd` |
| Sitemap | `renderSitemapXml`, `postsToSitemap`, `termsToSitemap` |
